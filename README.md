# Paseo Agent Demo

A scaffold for agentic projects using OpenRouter and Opencode.

## Directory Structure
- `src/`: Core agent code.
- `tests/`: Project tests.
- `docs/`: Project documentation.
- `examples/`: Sample usage scripts.
- `paseo/`: Docker setup and helper tools.

## Setup
1. Copy `.env.example` to `.env`.
2. Run `./paseo/opencode-tool setup-env`.
3. Start the service using `docker-compose up`.
